## <a id="tech-point">技术点</a>

 - 使用 `webpack` 实现项目模块化打包构建
 - 采用**响应式布局**，使项目可以在移动端以及PC端良好显示
 - 使用 `Bootstrap` 实现响应式布局
 - `CSS` 部分采用 `SCSS` 预处理器实现；对于项目中 `DOM` 的操作采用 `jQuery` 库
